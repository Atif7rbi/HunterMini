# HunterMini Experiments

HunterMini should run one major experiment at a time.

## Experiment Record

Each experiment must define:

- ID, for example `HM-001`
- Title
- Goal
- Exact change
- Start date
- Success criteria
- Final decision: promote, continue, or reject
