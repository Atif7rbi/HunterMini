# EXP-001 — Aggressive Profile Optimization

## Goal

Test improved aggressive trade management profile inside HunterMini before moving any proven change to HunterBot.

## Scope

Allowed:

- TP adjustment
- SL adjustment
- Break Even adjustment
- Trailing adjustment
- Profile tuning
- Report improvements

Not allowed:

- New strategy
- New indicators
- Direction Authority changes
- Mixing multiple experiments

## Promotion Criteria

- Minimum 100 trades
- Win Rate >= HunterBot
- Average R >= HunterBot
- Profit Factor >= HunterBot
- Drawdown <= HunterBot
- No critical bugs
- Reports complete
